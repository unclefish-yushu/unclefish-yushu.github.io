一文读懂NFT的玩法和分类

December 13, 2021

hello，大家好，我是鱼叔，最近我对NFT有很多的思考，本文作为我近期对NFT的一个思考总结。

NFT的分类

NFT的英文叫做nonfungible token，意思是不可分割的代币，这个其实很好理解，你可以买0.1个ETH，甚至0.01个ETH，但是你不能把蒙娜丽莎的画给撕碎了买其中的0.1个，所以说NFT的最大特点就是代币的完备性，无法像其他代币一样进行分割。

根据NFT的这个特性，它就衍生成为一种加密资产，可以理解为是加密世界的一种艺术品，并且它具有唯一性，因为每一个NFT的来源都是可以追溯的，相比于现实世界的假冒艺术品，NFT在虚拟世界里具有更高的防伪特征，这也是NFT艺术品被推崇的原因之一。 NFT由于其不可分割性，逐渐衍生成以下几个分类：

  1. NFT 艺术品
  2. NFT 身份标识
  3. GamFi - NFT游戏道具
  4. Real NFT - 和实体映射的NFT
  5. Other NFT - 互联网公司尝试的NFT

类别1 : NFT艺术品

NFT 艺术品是NFT最早的玩法，也是目前的主流玩法，它的商业模式和现实中的艺术品或者稀有品类似，都是通过拍卖收集的方式，不过加密世界NFT艺术品相比于现实世界中多了更多的自由度，通常在现实世界中普通人很难参与艺术品的设计和拍卖，但是在加密世界里，你可以自己铸造自己设计的艺术NFT，然后放到Opensea里去拍卖。目前较火的NFT艺术品是CryptoPunk，就是这种像素小人。

类别2 ：NFT 身份标识

由于NFT具有不可分割的特征，很多社区会给一些活跃的用户颁发NFT证书，这种NFT通常是一种身份标识，可以理解为是VIP会员卡。通常这种NFT身份不具备交易属性，第一很少人会去卖自己的身份奖励，第二这种身份NFT的发布没有上限。当然未来也有可能会以NFT艺术品作为身份标识，比如一个Crypto的社区要求成员拥有Cryptopunk才能加入，那么这种NFT即是艺术品，也是身份标识。

类别3 : GamFi

GamFi 是最近特别火的一个话题，原因是Axie Infinity 的Play to Earn 模式带动了区块链游戏的火爆。这类NFT通常是游戏里的道具，具有交易属性，拿一个现实世界里最有代表性的「梦幻西游」做比较，两者其实很类似，都具有内在的交易场，玩家都有机会通过游戏里的稀有道具转换成现实里的资产。GamFi 其实是一种NFT + Game + Defi的模式。

类别4 : RealNFT

RealNFT 是一种和现实世界里有直接映射关系的NFT，最有名的就是Uni早期出的袜子NFT，每一个NFT都对应着一双具体的袜子，你可以拿着NFT去兑换这双袜子，随后这个NFT就会被销毁，这种一一对应的特点就是RealNFT的玩法。再比如Sushi前一段时间合作推出的清酒SAKE NFT其实也是类似的，每一个NFT都可以兑换一瓶清酒，通常这类NFT的数量都是有限的，从而让它具备炒作的潜力。

当然这类RealNFT的玩法还有很多，包括有的人把自己房子所有权做成了NFT在加密世界上进行交易，也是这种方式的体现。

类别5 : OtherNFT

其他类的NFT我这里主要指一些互联网大厂发的NFT，比如支付宝的皮肤NFT，腾讯的音乐NFT，这类NFT通常是采用了NFT的技术但是不具备NFT的交易属性，也就是说你买了一个皮肤NFT，你是不能转手卖的，这类NFT不具备炒作潜力，只有收藏价值。

NFT的Defi+玩法

NFT + Defi 是将NFT的可能性进一步拓展的玩法。

1. 把NFT当作LP Token

将NFT作为一种质押凭证，是NFT和Defi的一次创新性合作，其中比较有代表性的是Uniswap V3 和 Aavegotchi。

Uniswap V3

uniswap v3将自己的LP token以一种NFT的方式呈现，由于NFT是一个多维度的画面，它可以将更加丰富的信息体现在里面，比如流动性信息、ID号、最初投入的代币占比等。但注意的是，这个NFT是赎回流动性的凭证，如果NFT被卖掉后，那么流动性也就无法拿回，现实里真的有人不小心把自己的LP NFT给卖了然后造成了损失。

Aavegotchi

Aavegotchi 是Aave投资的一个创新公司开发的项目，它是依赖于Aave的aToken然后生成的NFT，简单来讲就是你在Aave提供流动性得到aToken，然后再将aToken锁在Aavegotchi里来随机获得幽灵NFT，并且这个幽灵NFT的价值来源于你质押的aToken价值加上NFT的稀有度价值。

这种NFT的玩法和Axie Infinity 类似，也是通过购买道具或者升级来提高稀有度，但是不同的是每一个Aavegotchi NFT背后都是有一个aToken质押在里面的，如果要赎回aToken，那么对应的NFT也要被销毁，和Uni V3类似。

2. NFT 碎片化 （F-NFT）

虽然说NFT是不可分割的，但是有时候一个NFT特别贵，不是个体用户能参加的，为了降低购买NFT的成本，有的平台就将NFT碎片化（Fractional NFT），简单流程就是平台购买一个NFT，然后基于这个NFT的价值发放这个NFT的token，比如100个，每一个token就是原NFT价值的1/100，那么一些普通用户就可以买上0.01个NFT，如果你拥有全部的Token则可以直接置换这个完整的NFT。

这类F-NFT的代表比如之前提到的清酒SAKE，它就支持将NFT碎片化，当你集齐了一个完整的NFT，就可以去兑换一瓶清酒，如果你只是为了参与这个SAKE的增长，可以只买0.1个SAKE。

F-NFT 还算NFT吗？

在我看来，F-NFT只能算是NFT的一种金融衍生品，其实就是一种NFT的证券化方式，F-NFT的每一个token拥有者都不具备这个NFT的完整所有权。

3. NFT作为借贷的质押品

NFT的价格浮动很大而且流动性不是很强，尤其是艺术品类NFT，很多NFT的持有者，比如Cryptopunk持有者，会在需要资金的时候选择质押NFT来借贷而不是出售NFT，目前已经有平台支持NFT借贷比如Cream。当我们考虑NFT作为质押品时，我们需要知道NFT的真实价值到底是多少，而这个喂价一直是个难点，我会根据两种交易形式进行讲解。

1）质押给平台

如果交易对象是平台的话，这就涉及到DAO的投票了，比如说你要把Cryptopunk质押给Cream，目前市场价格是3000ETH，你想借1000ETH，看起来这个借贷风险不高，不过这笔交易需要整个DAO投票同意才可以，这个模式很难进行智能合约化。

2）质押给个人（P2P）

P2P的模式我认为是NFT质押的一个比较合理的方式，有点像“一个愿打一个愿挨”，因为NFT的价值其实很难估计，如果你把价值1000ETH的NFT挂了个300ETH的借贷单，那么有用户认为这个NFT的价值高于300ETH，他就会愿意借钱给你，因为对于他来说即使被清算拿到了这个NFT，他也可以凭借高于300ETH的价格卖出。

NFT借贷的风险

NFT借贷的最大风险来源于价格衡量没有标准，尤其是艺术品类NFT。通常一个NFT的价格是用挂单拍卖的方式炒作上去的，存在一种可能就是NFT所有者自己挂单自己买，逐渐把价格抬上去，然后在NFT价格较高的时候抵押借贷来套现，因此NFT借贷最大的问题还是NFT价值的判断。

未来的展望

NFT和DEFI结合必然是未来的趋势，纯粹立足于欣赏层面的NFT会逐渐泡沫化，因为审美这种东西是会随时代变化的，但是不变的是艺术品背后的故事，为什么大家喜欢梵高的向日葵，喜欢莫奈的日出印象，原因在于这些画的背后有历史，有故事，但是加密艺术品目前来看很多都缺乏积淀就草草上架，一波风潮后就剩下一地鸡毛。要么立足于故事，要么立足于实用性，否则NFT就将是空气。
